# Icon

All of the possible values for the icon come from the [material design icons](https://api.flutter.dev/flutter/material/Icons-class.html#constants).

## Examples

`value` just represent the id of the material design icons you want to use, check the doc to find one that fit your needs.

### Simple icon

```json
{
    "type": "icon",
    "value": "abc"
}
```

### Icon color & size

```json
{
    "type": "icon",
    "value": "abc",
    "color": 0xFF000000,
    "size": 100
}
```
