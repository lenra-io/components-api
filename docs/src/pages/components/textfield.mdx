# Textfield

This component gives the possibility to the user to write text.

## Examples

### Basic textfield
The `onChanged` event will be triggered if the user type (or remove) some text in the field.
So you can save it to your data and react to any user input later.

```json
{
    "type": "textfield",
    "value": "fixedValue",
    "onChanged": {
        // This listener cannot change the value because the value is fixed to "fixedValue"
        "action": "changeValue"
    }
}
```


### Complex textfield
Like some other components (`container`, ...), `textfield` allow you to change its style using the `decoration` property.
But as you can see, it's really easy to react to the user input on the fly.

```json
{
    "type": "textfield",
    // The value is stored in the app data
    "value": data,
    "style": {
        "decoration": {
            "icon": {
                "type": "icon",
                "value": "star",
            },
            "filled": true,
            "fillColor": 0xFFBBDEFB,
            "border": {
                "type": "outline",
            },
            "hintText": "hint",
            "helperText": "helper",
            "labelText": "label",
            // Showing the string length under the textfield
            "counterText": "${data.length}"
        },
    },
    "minLines": 3,
    "maxLines": 5,
    // This listener takes care of changing the value in the app data
    "onChanged": {"code": "CodeA"}
},
```
