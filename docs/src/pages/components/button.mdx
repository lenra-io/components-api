This is the button component.

You can set its text, disable it, adjust its size and style, react to a click and add left or right icons.

## Examples

### Simple button

<CodeBlock>
```json
{
    "type": "button",
    "text": "My button",
    "onPressed": {
        "action": "buttonPressed"
    },
}
```
</CodeBlock>

### Disabled button

<CodeBlock>
```json
{
    "type": "button",
    "text": "My button",
    "disabled": true,
}
```
</CodeBlock>

### Size and style adjusted button

<CodeBlock>
```json
{
    "type": "button",
    "text": "My button",
    "size": "large",
    "mainStyle": "secondary",
}
```
</CodeBlock>

### Button with icons

<CodeBlock>
```json
{
    "type": "button",
    "text": "My button",
    "leftIcon": {
        "type": "icon",
        "value": "smart_button",
    },
    "rightIcon": {
        // You can also use a view instead of an icon
        "type": "text",
        "value": "This can be used as an icon"
    },
}
```
</CodeBlock>
